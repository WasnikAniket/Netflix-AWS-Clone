# 🎬 Netflix AWS Clone

A serverless Netflix-style movie management application built using AWS.

## 🚀 Features

- Upload movie details
- Store movie information in Amazon DynamoDB
- Retrieve all movies using REST API
- Serverless backend using AWS Lambda
- API Gateway integration
- Responsive frontend with HTML, CSS and JavaScript

---

## 🛠 Technologies Used

- AWS Lambda
- Amazon API Gateway
- Amazon DynamoDB
- HTML5
- CSS3
- JavaScript
- GitHub
